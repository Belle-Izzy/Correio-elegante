# Correio-elegante
Um site feito inteiramente por mim, para automatizar a criação das cartas do correio elegante que minha turma do terceiro ano do ensino médio gostaria de fazer

## Como funciona:
- O usuário entra no site http://correio-elegante-tercetroia.free.nf
- Clica em "Enviar Cartinha"
- Seleciona a frase que gostaria de colocar e preeche as informações
  - **Caso o usuário queria como anônimo ele seleciona o quadradinho "Desejo não informar" no bloco que irá deixar em branco na cartinha**
  - **O número máximo de caracteres nos nomes é de 22**
- Depois de clicar em "Gerar Cartinha" o sistema carrega o modelo referente a frase que escolheu e gera um novo bilhete (.png) de nome aleatório (opção de código com nome numerado comentado) na pasta referente a turma que ele vai mandar
  - **Modelos criados também por mim na pasta assets/modelos (5,5cm x 5cm)**
